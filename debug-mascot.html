Use esta página para debugar problemas com o mascote:

Instruções:

  1. Clique em "Corrigir e Ir para Site" para limpar localStorage e ir ao site
  2. Alternativamente, abra o Console do navegador (F12) no site e execute:
// Limpar localStorage e forçar mascote visible
localStorage.removeItem('crset_show_mascot');
localStorage.setItem('crset_show_mascot', 'true');
location.reload(); // Recarregar página
